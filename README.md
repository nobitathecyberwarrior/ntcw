# NTCW - Nobita The Cyber Warrior Portfolio

This is the official animated portfolio website for **Nobita The Cyber Warrior (NTCW)**, a dedicated cybersecurity agency founded by Md Yousuf Ali.

## Features
* Hacker-themed UI with Matrix Rain animation
* Glitch text effects
* Linux Terminal style 'About Us' section
* Fully responsive design

## Technologies Used
* HTML5
* CSS3
* JavaScript (Vanilla)
